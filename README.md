A **cyberpunk-inspired personal portfolio** floating in space — built with pure HTML, CSS, and vanilla JavaScript. Features animated starfield background, neon glow effects, floating profile, typing animation, interactive skill cards, and smooth responsive design.

**Live Demo** 🚀:  
https://ibrahimowolabiyomide-rgb.github.io/Zentarolbrahim/

(Once you rename **My portfolio.html** → **index.html**, the URL loads cleaner without the filename in it.)

## 🌠 Features
- Infinite animated space background with moving stars (canvas + requestAnimationFrame)
- Neon gradient glowing name with shifting animation
- Typed.js-style intro text animation
- Floating profile picture with hover/float effect
- Interactive skills grid — click to reveal details
- Responsive layout (mobile-first) with smooth scroll
- Fixed blurred navbar with cyan hover effects
- Contact icons with direct links (email, WhatsApp, X)

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **CSS3** – Flexbox/Grid, animations, gradients, backdrop-filter blur
- **JavaScript (Vanilla)** – Typing effect, skill toggle, starfield animation
- **Google Fonts** – Orbitron (futuristic sci-fi vibe)
- **Font Awesome** – Icons for contact
- **CDN** – Lightweight, no build tools needed
